# 🌐 Network Anomaly Detector

A Python-based real-time network monitoring and anomaly detection system that uses machine learning to identify suspicious network behavior using traffic statistics and Isolation Forest.

---

## 🚀 Features

- Real-time packet sniffing using Scapy
- Network traffic statistics monitoring
- Protocol analysis (TCP, UDP, ICMP)
- Source & destination tracking
- Machine learning-based anomaly detection (Isolation Forest)
- Live terminal dashboard
- Automatic CSV logging of network activity
- Alert system for suspicious behavior

---

## 🛠️ Tech Stack

- Python
- Scapy (packet inspection)
- NumPy & Pandas (data processing)
- Scikit-learn (Isolation Forest ML model)
- Rich (live terminal UI)
